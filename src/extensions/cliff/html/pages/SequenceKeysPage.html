Key Sequencer Tips

  • Manually enter your master or sub-master key(s) into "Records->Keys".
  • Only use the sequencer for a single level and branch of an MK system, the 'Stamp' will be statically assigned to all keys
  • When generating keys in a branch (or beneath a master key), only enter multiple values for the chambers that should vary at this level of the system.
  • Remember that A2 chambers are ideally set up odd-over-odd or even-over-even.
  • VERIFY your key codes by building a core and reading it. Ensure there is no key-crossover.
